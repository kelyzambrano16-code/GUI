Este proyecto en pycharm trabaja con la libreria PySide6 y Qt designer para crear un programa con una ventana de interfaz gráfica que permite el ingreso de información para guardar en una base de datos cuenta con directorios
que son los siguientes:

1. Datos Encargada de conectar con la base de datos y afines.
2. Servicio Encargada de la validación de los datos que ingrese el usuario.
3. Dominio Encargada de modelar y encapsular la información del usuario mediante atributos y propiedades, facilitando su transporte entre la interfaz y la lógica.
4. Ui Contiene los archivos de la interfaz como vtnPrincipal.py, Interfaz.ui.
5. Main encargado de ejucutarla.
